# רשימת בדיקות ידניות מקיפה - OmerOpsMap Frontend

> **600+ בדיקות ממצות** לכל פונקציונליות במערכת

---

## הכנה ראשונית

### 1. וידוא שהמערכת רצה

```powershell
# בדוק שכל הcontainers רצים
docker ps

# צריך לראות 3 containers:
# - omeropsmap_postgres
# - omeropsmap_data_server
# - omeropsmap_frontend
```

### 2. פתח את האפליקציה

- URL: http://localhost:5173
- דפדפן מומלץ: Chrome/Edge (F12 פתוח לקונסול)
- רזולוציות לבדיקה: Desktop (1920x1080) + Mobile (375x667)

### 3. נקה sessionStorage לפני התחלה

```javascript
// בקונסול F12:
sessionStorage.clear()
location.reload()
```
